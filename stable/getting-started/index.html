Getting Started with Minterpy#

Have you installed Minterpy? Read on how to install it before moving on.

Installation#

The public release of Minterpy (i.e., the main branch) can be obtained directly from PyPI with pip:

pip install minterpy

The latest development branch of Minterpy can be obtained from its GitHub repository:

git clone -b dev https://github.com/casus/minterpy

After moving inside the cloned directory, the package can be installed from source with pip:

pip install [-e] .

A best practice is to create a virtual environment so as not to install external package to the your base Python environment. You can do this with the help of, among others:

  • venv

  • virtualenv

  • mamba

  • conda

Quick Start Guide to Function Approximations with Minterpy

While approximating functions using polynomials is a main feature of Minterpy, it also offers multi-dimensional polynomials in Python. These polynomials have a consistent interface that allows for advanced manipulation such arithmetic and calculus operations.
