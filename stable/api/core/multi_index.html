class minterpy.core.multi_index.MultiIndexSet(exponents: ndarray, lp_degree=None)#

Bases: object

Class representation of the set of multi indices for a polynomial.

The instances of this class provide storrage for the exponents of a multi variate polynomial independently of the base assumed in the polynomial space. Only the polynomial degree (w.r.t. a given l_p norm) as well as the dimension of the space are used to build the set of exponents.

Attributes:
exponents

Array of exponents.

lp_degree

The degree for the \(l_p\)-norm used to define the polynomial degree.

The completed version of the exponent array.

property exponents#

Array of exponents.

Returns:

A 2D ndarray which stores the exponents, where the first axis refers to the explicit exponent and the second axis refers to the variable which is powerd by the entry at this array position. These exponents are always lexicographically ordered w.r.t. the first axis.

Return type:

np.ndarray

property exponents_completed#

The completed version of the exponent array.

Returns:

A complete array of the exponents, i.e. in lexicographical ordering, every exponent which is missing in exponents, will be filled with the right entry.

Return type:

np.ndarray
