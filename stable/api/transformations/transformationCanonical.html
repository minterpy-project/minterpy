From Canonical basis#

Concrete implementations of the Transformation classes for the CanonicalPolynomial.

Transformations from Canonical basis to Newton and Lagrange basis are provided.

class minterpy.transformations.canonical.CanonicalToNewton(origin_poly: MultivariatePolynomialSingleABC)#

Bases: TransformationABC

Transformation from CanonicalPolynomial to NewtonPolynomial

__call__(origin_poly: MultivariatePolynomialSingleABC | None = None)#

Transformation of polynomial.

This function is called when an instance of transformation is called T() or T(x). If called without any arguments, the transformation is applied on the origin_poly used to construct the transformation. If an argument is passed, the transformation is applied on the instance passed.

Parameters:

origin_poly – (optional) an instance of the polynomial to be transformed.

Returns:

an instance of the polynomial in the target basis.
