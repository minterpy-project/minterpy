Interpolation in unisolvent nodes#

Generating nodes#

The generating nodes are given by choosing arbitrary sets \(P_i \subseteq [-1,1]\) of \(n_i=|P_i| \in \mathbb{N}\) nodes for each dimension \(m \in \mathbb{N}\) such that for fixed multi-index set \(A\subseteq \mathbb{N}\) there holds \(n_i \geq \max \{\alpha_i : \alpha \in A\}\).

Whenever \(A = A_{m,n,p}\) is given by a multi-index set of specified degree then by default the nodes are chosen to be the Chebyshev-Lobatto nodes

(1)#\[P_i =\{p_{0,i},\dots,p_{n,i}\} = (-1)^m \mathrm{Cheb}_n = \left\{ \cos(k\pi/n) : 0 \leq k \leq n\right\}\]

Apart from equidistant nodes other prominent choices of generating nodes are given by the Gauss-Legendre nodes, Chebyshev nodes of first & second kind, equidistant nodes, etc., see e.g.[1][2][3]. For treating the generating nodes in a compactified way they are implemented as an array

\[\mathrm{GP} = \oplus_{i=1}^m P_i\,.\]

Crucial is the choice of the ordering of the node sets \(P_i\). By default we assume that the generating nodes \(\mathrm{GP}\) are Leja-ordered[4], i.e, there holds:

\[|p_0| = \max_{p \in P}|p|\,, \quad \prod_{i=0}^{j-1}|p_j-p_i| = \max_{j\leq k\leq m} \prod_{i=0}^{j-1}|p_k-p_i|\,,\quad 1 \leq j \leq n\,.\]

The Leja ordering does not only guarantee numerical stability of the Newton interpolation realised in the multivariate DDS but also makes the resulting unisolvent nodes to be a (sparse, non-tensorial) grid of high approximation power.

Unisolvent nodes#

For multi-index set \(A \subseteq \mathbb{N}^m\) and chosen generating nodes \(\mathrm{GP} = \oplus_{i=1}^m P_i\) the unisolvent nodes are given as the sub-grid

\[P_A = \left\{ p_\alpha = (p_{\alpha_1,1},\ldots,p_{\alpha_m,m}) \in \Omega\subseteq \mathbb{R}^m : \alpha \in A\right\}\,, \quad p_{\alpha_i,i} \in P_i\,.\]

In addition to the possible different choices of the \(P_i\) in each dimension the impact of different orderings can be observed in the example below. Thereby, the \(P_i\) just differ in their orderings that are indicated by enumerations along the dimensions (\(x,y\)). The resulting unisolvent nodes may be given by non-tensorial or non-symmetric grids, where nodes exist with \(p=(p_x,p_y) \in P_A\), but \((p_y,p_x)\not \in P_A\).

Examples of unisolvent nodes in 2D & 3D for the default choice of Leja ordered Chebyshev-Lobatto nodes given in Eq. (1) are visulaised below.

From a general perspective, a deeper discussion of their construction and resulting properties is given in[5]. Most crucial is the fact that for downward closed multi-index sets \(A\subseteq \mathbb{N}^m\) the interpolant \(Q_{f,A}\) is uniquely determined in the polynomial space

\[\Pi_A =\left<x^\alpha = x_1^{\alpha_1}\cdots x_m^{\alpha_m} : \alpha \in A\right>\]

spanned by all canonical monimials.

Lagrange interpolation#

Given a set of multi-indices \(A \subseteq \mathbb{N}^m\,, m \in \mathbb{N}\), unsisolvent nodes \(P_A \subseteq \Omega = [-1,1]^m\) and a function \(f: \Omega\longrightarrow \mathbb{R}\) the uniquely determined Lagrange interpolant \(Q_{f,A}\) of \(f\) is given by

\[Q_{f,A}(x) = \sum_{\alpha \in A}f(p_{\alpha})L_{\alpha}(x)\,, \quad p_{\alpha} \in P_A\,,\]

where \(L_\alpha\) denote the Lagrange polynomials satisfying \(L_{\alpha}(p_\beta) = \delta_{\alpha,\beta}\) with \(\delta_{\cdot,\cdot}\) denoting the Kronecker delta. In fact, deriving the Lagrange interpolant \(Q_{f,A}\) of a function \(f\) is trivial and can be executed with linear \(\mathcal{O}(|A|)\) storage amount and runtime complexity whenever the unisolvent nodes \(P_A\) are given. However, Lagrange interpolation is a rather theoretical concept while it does not provide a computational scheme for evaluating the interpolant \(Q_{f,A}\) in any given argument \(y \in \Omega\).

Newton interpolation#

In contrast to Lagrange interpolation, Newton interpolation derives the uniquely determined Newton interpolant \(Q_{f,A}\) of \(f\) given by

\[Q_{f,A}(x) = \sum_{\alpha \in A}c_\alpha N_{\alpha}(x)\,, \quad \alpha \in A\,,\]

where \(A \subseteq \mathbb{N}^m\) is a set of multi-indices, \(P_A \subseteq \Omega = [-1,1]^m\) denotes the unsisolvent nodes and \(f: \Omega\longrightarrow \mathbb{R}\) a function.

Thereby, the coefficients \(c_\alpha \in \mathbb{R}\) of the Newton polynomials

\[N_\alpha(x) = \prod_{i=1}^m\prod_{j=0}^{\alpha_i -1}(x- p_{j,i})\,,\quad p_{j,i} \in P_i\]

can be derived by the multivariate Divided Difference Scheme (DDS) whenever the generating nodes \(\mathrm{GP} = \oplus_{i=1}^m P_i\) are fixed. In fact, DDS requires quadratic \(\mathcal{O}(|A|^2)\) runtime complexity and linear storage \(\mathcal{O}(|A|)\) amount.

Once given the Newton interpolant of a function \(f\) its evaluation can be realised by a generalisation of the classic Aitken-Neville algorithm[6].
